Website

It countains several webpages and data files that internet users can access through a browser.

WWW

It refers to all the public websites or pages that users can access on their local computers and other devices through the internet.

HTTP

Hyper Text Transfer Protocol, it is a application-level protocol used for exchanging files on the WWW.

Client Server

The client request a service or resourse from one program to another program to the server.

Domain

It is a group of connected windows in computers that share user account information and a security policy.

Host

Any hardware device that has the capacity of permitting access to a network through a user interface, network address, or any other means.

Hosting

When a hosting provider allocates space or a web server for a website to store its files.

FTP

It is a standard network protocol used for the transfer of files from one host to another over a transmission control protocol based network such as internet.

Path

A string of character used to uniquely identify a location in a directory structure.

Extension

Are code packages that run inside visual studio and provide new or improved features.

Editor/VScode

It is a source code editor made by microsoft with the electron framework, for windows, linux and macOS.

URL-URI/URN

URL is unique identifier used to locate a resource on the internet.

Address bar

It is the text field at the top of a web browser's that displays the name or URL of the current web page.

Title bar

It is located at the topmost part of a window within grafical user interface. It shows the software title that is currently running.
